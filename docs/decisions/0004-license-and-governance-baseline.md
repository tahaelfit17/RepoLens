# 0004: License and governance baseline

Status: Accepted

## Context

RepoLens depends on trust. The project should avoid governance choices that make
users fear vendor capture or future source closure.

## Decision

RepoLens uses the Apache License 2.0 and the Developer Certificate of Origin.
The project starts without a Contributor License Agreement.

## Consequences

- Users receive a permissive license with an explicit patent grant.
- Contributors sign off commits instead of assigning broad rights through a CLA.
- Any future governance or foundation move must be documented in a new design
  decision before it happens.
